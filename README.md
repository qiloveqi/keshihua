## 一、技术栈

该项目主要使用 `Vue`搭配`Echarts`采用组件化的方式开发完成,此外项目中陆续使用有

- VueRouter
- VueX
- WebSocket
- axios
- webpack
- ES6+
- less

## 二、如何运行该项目？

```js
// 首先你应当克隆该项目
git clone https://gitee.com/penggang-home/data-view.git

// 进入该项目
cd data-view

// 安装对应依赖
cnpm install

// 访问线上后台系统
npm run serve
```
